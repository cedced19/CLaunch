### Description
Marche depuis **Minecraft 1.0**

Modifier le **server.properties**:

> *enable-query=true*  
> *query.port=25566*  
> *enable-rcon=true*  
> *rcon.port=25567*  
> *rcon.password=ichbinpassword*
